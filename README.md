# Java Selenium E-cart App Automation
## OVERVIEW:
Automated test suite for E-cart application using Java, Selenium WebDriver, TestNG, POM and Maven.

## FEATURES:
User authentication  to the application

Selects products dynamically

Adds items to Cart

Shows Orders checkout details

Completes checkout

Uses TestNG for framework development

Incorporates Page Object Model for reusable code

Automatically captures Screenshot under test failure

## PREREQUISITES:
Java 25

Chrome

Selenium, Maven Setup

## RUN:

HTML Report generation, Extent reports

STRUCTURE:
Page Object Model by using TestNG methods and page factory concepts

Data driven approach for dynamic user inputs
